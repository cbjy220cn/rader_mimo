┌─────────────────────────────────────────────────────────────────┐
│                        实验说明                                 │
├─────────────────────────────────────────────────────────────────┤
│ 【实验目的】                                                    │
│   验证运动合成孔径DOA算法对平台振动的鲁棒性。                   │
│   无人机飞行时存在振动，导致实际位置与理论轨迹存在偏差。        │
│                                                                 │
│ 【振动模型】                                                    │
│   实际位置 = 理论位置 + 高斯随机偏差                           │
│   偏差 ~ N(0, σ²)，σ为振动标准差，单位为波长λ                  │
│   各时刻偏差独立，同一时刻所有阵元偏差相同（整体振动）          │
│                                                                 │
│ 【关键指标定义】                                                │
│   ① RMSE (均方根误差)                                          │
│      = sqrt(mean((估计值-真实值)²))                            │
│      物理含义：估计精度的综合度量，越小越好                     │
│                                                                 │
│   ② 偏差 (Bias)                                                │
│      = mean(估计值-真实值)                                     │
│      物理含义：系统性偏移，反映估计是否存在系统误差             │
│                                                                 │
│   ③ 标准差 (Std)                                               │
│      = std(估计值-真实值)                                      │
│      物理含义：估计的离散程度，反映一致性                       │
│      关系：RMSE² = Bias² + Std²                                │
│                                                                 │
│   ④ 成功率                                                     │
│      = (|误差|<阈值的次数) / 总试验次数 × 100%                 │
│      物理含义：系统可靠性指标                                   │
│      - 100%: 所有情况准确，系统完全可靠                        │
│      -  90%: 10次有1次失败，可接受                             │
│      -  50%: 一半失败，临界状态                                │
│      - <50%: 系统不可靠                                        │
│                                                                 │
│   ⑤ 中位绝对误差                                               │
│      = median(|估计值-真实值|)                                 │
│      物理含义：对离群值鲁棒的精度指标                           │
│                                                                 │
│ 【实际应用参考】                                                │
│   典型无人机振动RMS：0.1-1 cm                                  │
│   若λ=10cm(3GHz)，则振动≈0.01-0.1λ                            │
└─────────────────────────────────────────────────────────────────┘
