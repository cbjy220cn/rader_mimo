│ 【关键指标定义】                                                │

│                                                                 │
│   ② 分辨率                                                     │

│                                                                 │
│   ③ 谱峰峰值比                                                 │
│      = 目标峰 / 最大旁瓣                                       │
│      物理含义：目标检测的可靠性                                 │
│                                                                 │
│   ④ 峰值位置误差                                               │

│                                                                 │
│   ⑤ 3dB主瓣宽度                                                │
│      = 谱峰下降3dB对应的角度范围                               │
│      物理含义：与分辨率直接相关，主瓣越窄分辨率越高             │
│                                                                 │
│ 【实验设计】                                                    │
│   对比静态阵列与运动阵列在不同角度间隔下的分辨能力              │
