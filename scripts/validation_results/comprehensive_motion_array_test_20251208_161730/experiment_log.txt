┌─────────────────────────────────────────────────────────────────┐
│                        实验说明                                 │
├─────────────────────────────────────────────────────────────────┤
│ 【实验目的】                                                    │
│   全面评估运动合成孔径DOA算法在不同阵列形状、运动模式、         │
│   SNR条件下的性能表现，与静态阵列进行对比。                     │
│                                                                 │
│ 【核心原理】                                                    │
│   运动合成孔径：利用平台运动扩展虚拟阵列孔径                    │
│   - 静态N元阵列：孔径 ≈ (N-1)d                                 │
│   - 运动后孔径：孔径 = (N-1)d + 运动距离                       │
│   孔径扩大 → 角度分辨率提高 → DOA精度提升                      │
│                                                                 │
│ 【关键指标定义】                                                │
│   ① RMSE (均方根误差)                                          │
│      = sqrt(mean((估计角度-真实角度)²))                        │
│      单位：度(°)                                               │
│      物理含义：DOA估计精度的综合度量                            │
│      典型值：静态阵列0.1-1°，运动阵列0.05-0.2°                 │
│                                                                 │
│   ② 合成孔径 (Synthetic Aperture)                              │
│      = 虚拟阵列最大空间跨度 / λ                                │
│      单位：波长(λ)                                             │
│      物理含义：等效阵列的有效口径                               │
│      关系：孔径越大 → 分辨率越高 → RMSE越低                    │
│                                                                 │
│   ③ 3dB主瓣宽度 (Beamwidth)                                    │
│      = MUSIC谱峰下降3dB对应的角度范围                          │
│      单位：度(°)                                               │
│      物理含义：角度分辨能力的直接体现                           │
│      近似公式：θ_3dB ≈ 50.8°/(孔径/λ)                         │
│                                                                 │
│   ④ 改善倍数                                                   │
│      = RMSE_静态 / RMSE_运动                                   │
│      物理含义：运动带来的精度提升                               │
│      典型值：平移运动可达10-100倍改善                          │
│                                                                 │
│ 【运动模式说明】                                                │
│   静态：阵列不动，作为对照基准                                  │
│   x平移：沿x轴直线运动，扩展y方向孔径                          │
│   y平移：沿y轴直线运动，扩展x方向孔径                          │
│   旋转：原地旋转，不扩展孔径（对照组）                          │
│   平移+旋转：复合运动，同时扩展孔径并改变观测角                 │
│                                                                 │
│ 【阵列形状说明】                                                │
│   ULA: 均匀线阵，结构简单，单方向分辨                          │
│   URA: 均匀矩形阵，可二维分辨                                  │
│   圆阵: 各向同性，适合全向探测                                 │
│   L/T/Y/十字: 稀疏阵列，孔径大但阵元少                         │
│                                                                 │
│ 【结果解读】                                                    │
│   - 平移运动显著优于静态和纯旋转                               │
│   - 纯旋转不扩展孔径，性能与静态相近                           │
│   - 孔径扩展倍数约等于RMSE改善倍数                             │
└─────────────────────────────────────────────────────────────────┘

  4. 旋转
