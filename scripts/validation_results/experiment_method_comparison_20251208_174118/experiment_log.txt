├─────────────────────────────────────────────────────────────────┤
│ 【实验目的】                                                    │
│   对比两种合成孔径DOA估计方法的精度和特性                       │
│                                                                 │
│ 【方法1: CSA-BF (相干合成孔径波束形成)】                        │
│   原理：M×K虚拟阵列 + 匹配滤波                                 │
│   公式：P(θ) = |a(θ)'x|² / |a(θ)|²                            │
│   优势：完全相干利用孔径，分辨率高                              │
│   劣势：对相位误差敏感，无超分辨能力                            │
│                                                                 │
│ 【方法2: ISA-MUSIC (非相干合成孔径MUSIC)】                      │
│   原理：分段MUSIC + 谱累加                                     │
│   公式：P(θ) = Σ P_seg(θ)                                      │
│   优势：对相位误差鲁棒，有超分辨能力                            │
│   劣势：未完全相干利用孔径                                      │
│                                                                 │
│ 【方法3: 标准MUSIC (静态基准)】                                 │
│   原理：标准MUSIC，不利用运动                                  │
│   用于对比运动带来的增益                                        │
│                                                                 │
│ 【评估指标】                                                    │
│   ① RMSE: 均方根误差                                           │
│   ② 3dB主瓣宽度: 角度分辨率                                    │
│   ③ 计算时间: 算法效率                                         │
└─────────────────────────────────────────────────────────────────┘
